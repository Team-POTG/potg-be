# POTG Typing Rule

| 규칙                                                 | 올바른 예시           | 잘못된 예시           |
| ---------------------------------------------------- | --------------------- | --------------------- |
| 클래스, 상수를 제외한 모든 이름은 소문자로 시작한다. | getFoo                | GetFoo                |
| 각 단어마다 대문자는 하나로 통일한다.                | getApi</br>AccountDto | getAPI</br>AccountDTO |
